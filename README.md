# AI Automation Workflows Portfolio

This repository contains real-world automation workflows built using **Make.com** and **n8n**, focusing on AI-powered business automation.

- AI-powered email responses
- SEO blog generation
- WordPress automation
- Telegram bot integrations
- Landing page analysis
- Webhook data pipelines

## Platforms Used
- Make.com
- n8n
- OpenAI API
- WordPress
- Trello
- Telegram
- Zapier

This repository serves as a portfolio demonstrating automation design, AI agent integration, and workflow documentation.
