# Capture Custom Webhooks and Store in Database

## Overview
This workflow captures incoming webhook data and stores it in a database.

## Tools Used
- Make.com
- Custom Webhooks
- Database (PostgreSQL / MySQL / Airtable)

## Workflow Steps
1. Webhook receives data
2. Payload is validated
3. Data is transformed
4. Data is stored in database

## Business Use Case
Centralizes data collection from external services.
